# Spice Palace — Frontend Prototype

Premium-quality multi-page frontend for a Restaurant Management System built with Bootstrap 5.3, jQuery, and lightweight free libraries.

Features
- 100% responsive design (mobile/tablet/desktop)
- Dark & Light mode with localStorage persistence
- Glassmorphism, Neumorphism and gradient accents
- Preloader and smooth animations with AOS
- Hero with parallax effect
- Menu with search, category filter, add-to-cart animation, and offcanvas cart
- Reservation form with Flatpickr date/time pickers
- Gallery with GLightbox
- Testimonials with Swiper
- Admin dashboard demo using Chart.js
- Toastify notifications, Floating WhatsApp & Call buttons, Back-to-top

Pages included
- `index.html` — Home
- `menu.html` — Full menu
- `gallery.html` — Gallery
- `reservation.html` — Reservation form
- `checkout.html` — Checkout demo
- `blog.html` — Blog list
- `dashboard.html` — Admin demo

How to use
1. Upload the folder to any static host (Netlify, Vercel, GitHub Pages).
2. Open `index.html` in a browser.
3. No build step required; libraries are loaded from CDNs.

Notes
- Images used are hotlinked from Unsplash for demo purposes — replace with production assets before publishing.
- All interactions are client-side demo implementations; integrate with backend APIs for production.
